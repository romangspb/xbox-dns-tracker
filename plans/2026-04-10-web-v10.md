## Challenge Log

### 1. Решает ли это проблему?

Все 9 AC покрыты фазами 1-3.

### 2. Самое эффективное решение?

**Альтернатива 1:** React/Vue SPA → overkill для 1 страницы, 5 пользователей. Отклонено.
**Альтернатива 2:** Vercel/Netlify → лишний сервис, GitHub Pages бесплатен. Отклонено.
**Альтернатива 3 (выбрана):** vanilla HTML/CSS/JS + GitHub Pages + Actions. Ноль зависимостей фронтенда.

### 3. Нет ли кода ради кода?

- Без сборщиков (webpack/vite) — один HTML, один CSS, один JS
- Без CSS-фреймворков — 80 строк CSS
- Без роутинга — одна страница

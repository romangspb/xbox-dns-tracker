Как пользоваться

Выбор DNS

Нажми на IP-адрес — он скопируется. Открой Настройки Xbox → Сеть → Дополнительные → DNS (вручную) и вставь.

Статусы проверки

Работает — DNS обходит блокировку Xbox

Небезопасен — обходит, но подменяет обычные сайты

Не отвечает — сервер не ответил (может быть недоступен из вашего региона)

Не обходит — DNS не помогает

Кружки пользователей

Нажми на свой кружок под DNS чтобы отметить: работает (зелёный), не работает (красный), не пробовал (серый).

Категории

DNS IPv4 — обычные DNS, прописываются в Xbox

DNS IPv6 — для сетей с IPv6

xsts IP — IP для подмены на роутере (продвинутый способ)

Предложить DNS

Нашёл рабочий DNS или источник? Нажми «+ Предложить DNS» внизу — откроется форма на GitHub.

После отправки DNS появится в очереди на проверку. Если он рабочий — добавится на сайт при следующем обновлении. Нужен GitHub-аккаунт.

Как пользоваться

Выбор DNS

Нажми на IP-адрес — он скопируется. Открой Настройки Xbox → Сеть → Дополнительные → DNS (вручную) и вставь.

Статусы проверки

Работает — DNS обходит блокировку Xbox

Небезопасен — обходит, но подменяет обычные сайты

Не отвечает — сервер не ответил (может быть недоступен из вашего региона)

Не обходит — DNS не помогает

Кружки пользователей

Нажми на свой кружок под DNS чтобы отметить: работает (зелёный), не работает (красный), не пробовал (серый).

Категории

DNS IPv4 — обычные DNS. Начинай с них — подходят всем.

DNS IPv6 — пробуй только если IPv4 не помогли и твой интернет поддерживает IPv6 (спроси провайдера или посмотри в настройках роутера). IPv6 DNS не проверяются автоматически — пробуй вручную.

xsts IP — IP для подмены на роутере (продвинутый способ, нужна админка роутера).

Проверка доступа (большая кнопка)

Кнопка «Проверить мой доступ к Xbox» сверху страницы — одним нажатием проверяет, какие из всех путей обхода доступны именно из твоей сети.

Результат показывает сколько путей работают, и ставит метки на карточки:

✓ — вероятно сработает (путь доступен из твоей сети). Попробуй на Xbox.

✗ — точно не сработает (путь недоступен из твоей сети). Не трать время.

? — не хватило данных (IPv6 или ошибка проверки).

Важно: выключи VPN на время проверки — иначе результаты будут неверными.

Предложить DNS

Нашёл рабочий DNS или источник? Нажми «+ Предложить DNS» внизу — откроется форма на GitHub.

После отправки DNS появится в очереди на проверку. Если он рабочий — добавится на сайт при следующем обновлении. Нужен GitHub-аккаунт.

Выключи VPN

Сейчас мы проверим из твоей реальной сети, какие способы обхода Xbox работают.

Если у тебя включён VPN, проверка пойдёт через него, а не через твоего провайдера. Результаты будут неверные: VPN-туннель часто не видит российские прокси, хотя Xbox через обычный интернет их видит.

Важно: Xbox всё равно не умеет использовать system-wide VPN телефона — на консоль пойдёт прямое соединение через твой Wi-Fi/4G. Так что выключи VPN на телефоне только на время проверки.

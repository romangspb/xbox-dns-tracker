Как пользоваться

Выбор DNS

Нажми на IP-адрес — он скопируется. Открой Настройки Xbox → Сеть → Дополнительные → DNS (вручную) и вставь.

Статусы проверки

⚠️ Важно: статусы — это автоматическая проверка с нашего сервера. Это не гарантия что у тебя сработает — твой провайдер, регион или роутер могут блокировать конкретные IP. Просто пробуй сверху вниз.

Работает — DNS обходит блокировку Xbox (с нашего сервера)

Небезопасен — обходит блокировку, но подменяет обычные сайты (google.com и др.). Для Xbox подходит, но не оставляй такой DNS постоянно

Не отвечает — сервер не ответил с нашего инфраструктуры. Может работать у тебя — это часто Russian-only DNS, доступные только из РФ. Пробуй

Не обходит — DNS возвращает обычный IP Microsoft Azure, бесполезен для Xbox

Доступен — xsts IP отвечает на 443 порту с нашего сервера. Используется на роутере

Кружки пользователей

Нажми на свой кружок под DNS чтобы отметить: работает (зелёный), не работает (красный), не пробовал (серый).

Категории

DNS IPv4 — обычные DNS. Начинай с них — подходят всем.

DNS IPv6 — пробуй только если IPv4 не помогли и твой интернет поддерживает IPv6 (спроси провайдера или посмотри в настройках роутера). IPv6 DNS не проверяются автоматически — пробуй вручную.

xsts IP — IP для подмены домена xsts.auth.xboxlive.com на роутере (это домен авторизации Xbox, который Microsoft блокирует — поэтому ошибка 0x80a40401). Продвинутый способ, нужна админка роутера и поддержка hostname-override.

Предложить DNS

Нашёл рабочий DNS или источник? Нажми «+ Предложить DNS» внизу — откроется форма на GitHub.

После отправки DNS появится в очереди на проверку. Если он рабочий — добавится на сайт при следующем обновлении. Нужен GitHub-аккаунт.

Перед проверкой

1. Выключи VPN на устройстве. Сейчас проверка пойдёт через ту сеть, в которой открыт сайт. Если включён VPN — увидим, что проходит через VPN-туннель, а не через твою настоящую сеть. Xbox всё равно не использует system-wide VPN телефона, так что выключи на время проверки.

2. Если у тебя на роутере настроено правило для xsts (ты понимаешь о чём речь, если делал) — учти, что наша проверка покажет какие IP доступны именно из твоей сети. Чтобы это заработало на Xbox, рабочий IP нужно подставить в настройки роутера, а не на консоль — роутер всё равно перезапишет.
